API Documentation

POST /api/chat

Ask a question about the documents. Returns an AI-generated response with sources.

// Request { "question": "What is this document about?", "session_id": "optional-session-id" } // Response { "success": true, "answer": "The document is about...", "session_id": "session-id", "sources": ["document1.pdf"], "timestamp": "2023-10-05T12:00:00Z" }

GET /api/health

Check API status and document availability.

// Response { "status": "healthy", "timestamp": "2023-10-05T12:00:00Z", "documents_loaded": true, "documents_count": 5, "version": "2.0.0" }

GET /api/validate-key

Validate an API key. Requires Authorization header.

// Headers Authorization: Bearer your-api-key // Response { "success": true, "message": "API key is valid" }

GET /api/analytics

Get usage statistics and analytics about user interactions.

// Headers Authorization: Bearer your-api-key // Response { "total_queries": 150, "total_sessions": 25, "popular_questions": { "what is this about": 15, "summary": 12 }, "active_sessions": 3, "first_query": "2023-10-05T12:00:00Z", "last_query": "2023-10-05T14:30:00Z" }
